CSS PRE is your one-stop reference for the three most popular CSS Preprocessors - Less, Sass, and Stylus. The project is in its infancy, contribute on GitHub to make it better!

Syntax

The code examples for Sass on this site use the indented syntax, in the future SCSS syntax snippets may be added as well.

Stylus allows extreme freedom in terms of syntax, one can omit braces, semicolons, and even colons. To be future-proof, this site will only show Stylus snippets with colons (no semicolons or braces) and with a $ sign before each variable (which is currently optional) because this is how Stylus will most likely work in the future. I advise you to read Stylus' resources on what is allowed in terms of syntax.

Versions

This site is closely tied to CodePen, as a result all code is written to be compatible with the versions of the preprocessors that CodePen supports - Less, Sass, and Stylus.
